# desc_layout_ref

An explorer for describeLayout.  Go to http://localhost:8085/home, and log in with your username, password, and remote access key.  

## Usage

If you use cake, substitute 'lein' with 'cake' below. Everything should work fine.

```bash
lein deps
lein run
```
